Detailed Description

The ExportTypeSignalAction class Cette class herite de QAction qui permet d'envoie un signal qui signifie le type de export associe a cette action. Ceci permet d'utilise un seul Slot pour gerer un ensembles des actions (ex. ExportHtml, ExportTex etc...). Ici, nous avons utilise un pour rediriger la signal.
