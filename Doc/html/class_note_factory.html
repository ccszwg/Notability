Utilisation de horloge seul a precision d'un ms ne suffit pas pour le programme car los du changement de l'espace de travail, le temps que l'ordintateur copies les notes risque de prendre moins qu'un ms. On utilise ici une incrementation de note a note pour assurer unicite de identifiant des notes.
