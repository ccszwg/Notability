Detailed Description

The TreeItem class Cette classe est une implementation du model MVC propose par Qt MVC framework. Il s'agit d'un implementation personalisee de TreeItem. Cette classes sert a faciliter la gestion du TreeView. Chaque TreeItem comporte zero ou plusieurs childrens, qui sont a leurs tour potentiellement le parent des children. Ceci permet donc de former un arbre.
